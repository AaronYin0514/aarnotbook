# Git

## 克隆clone

```shell
git clone http://www.github.com/username/test.git
```

## 分支branch

### 创建分支

```shell
git branch dev
```

### 切换分支

```shell
git checkout dev
```

### 创建并切换

```shell
git checkout -b dev
```

## 标签Tag

### 切换到tag

```shell
git checkout tag_name
```
