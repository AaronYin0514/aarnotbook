# 模块

ES Module模块采用**export**和**import**关键字来实现模块化: 

- **export**：负责将模块内的内容导出;  
- **import**：负责从其他模块导入内容;

## export

 ### 方式一:在语句声明的前面直接加上export关键字

### 方式二:将所有需要导出的标识符，放到export后面的 {}中  
p 注意:这里的 {}里面不是ES6的对象字面量的增强写法，{}也不是表示一个对象的; p 所以: export {name: name}，是错误的写法;

### 方式三:导出时给标识符起一个别名
