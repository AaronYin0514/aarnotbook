## 创建项目

### 命令行

使用`flutter create`创建项目：

```shell
flutter create my_app
cd my_app
```

添加参数：

```shell
flutter create --project-name my_app --org dev.flutter --android-language java --ios-language objc my_app
cd my_app
```

运行App：

```shell
flutter run
```
