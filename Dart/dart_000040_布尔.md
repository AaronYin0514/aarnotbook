# 布尔类型

## 创建布尔类型变量

```dart
bool a = true;
bool b = false;
```

## 逻辑运算符

| 名称 | 逻辑运算符 |
| ---- | ---------- |
| 与   | `&&`       |
| 或   | `||`       |
| 非   | `!`        |

## 关系运算符

| 名称     | 关系运算符 |
| -------- | ---------- |
| 等于     | `==`       |
| 不等于   | `!=`       |
| 大于     | `>`        |
| 大于等于 | `>=`       |
| 小于     | `<`        |
| 小于等于 | `<=`       |
