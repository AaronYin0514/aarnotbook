# 张量-属性

## 属性

| 属性       | 说明    | 备注                                 |
| -------- | ----- | ---------------------------------- |
| `shape`  | 形状    |                                    |
| `ndim`   | 秩（维度） |                                    |
| `dtype`  | 数据类型  |                                    |
| `device` | 设备    | `cpu` `cuda` `mps`                 |
| `layout` | 内存布局  | `torch.strided` `torch.sparse_coo` |
