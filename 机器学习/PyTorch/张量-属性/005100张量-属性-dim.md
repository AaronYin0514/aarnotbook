## 张量-查看维度数-dim

`dim()` 方法用于返回张量的维度数量。

**示例**：

```python
import torch

# 创建一个3x4x5的张量
tensor = torch.randn(3, 4, 5)

# 获取张量的维度数量（秩）
num_dimensions = tensor.dim()

# 获取张量的形状
shape = tensor.size()

print(f"维度数量: {num_dimensions}")  # 输出: 维度数量: 3
print(f"张量的形状: {shape}")        # 输出: 张量的形状: torch.Size([3, 4, 5])
```
