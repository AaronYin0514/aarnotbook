# 张量-裁剪

对Tensor中的元素进行范围过滤。常用于梯度裁剪(gradient clipping)，即在发生梯度离散或者梯度爆炸时对梯度的处理。

```python
import torch

a = torch.rand(2, 2) * 10 # tensor([[2.9294, 6.9289], [9.3899, 1.1235]])

a = a.clamp(2, 5) # tensor([[2.9294, 5.0000], [5.0000, 2.0000]])

# 解释
# 限制元素范围 2 <= a <= 5
# 小于2 置为2 
# 大于5 置为5
```
