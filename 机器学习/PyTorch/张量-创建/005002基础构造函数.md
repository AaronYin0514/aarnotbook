## `torch.tensor`基础构造函数

```python
import torch

# 从列表创建一维张量
tensor_from_list = torch.tensor([1, 2, 3, 4])

# 从嵌套列表创建二维张量
tensor_from_nested_list = torch.tensor([[1, 2], [3, 4]])

print(tensor_from_list)
'''
tensor([1, 2, 3, 4])
'''

print(tensor_from_nested_list)
'''
tensor([[1, 2],
        [3, 4]])
'''
```
